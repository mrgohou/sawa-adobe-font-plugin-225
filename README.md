# Trouvez et installez vos polices directement dans vos calques
Sawa Font Finder est l'extension UXP officielle pour identifier, installer et apparier instantanément toutes les polices manquantes de vos documents au sein de Photoshop, Illustrator, After Effects, Premiere Pro, InDesign et InCopy.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
